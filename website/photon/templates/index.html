Photon search-as-you-type with OpenStreetMap

  • Made with OpenStreetMap

  • Open Source

  • Powered by SolR

A simple API

  • Search

    http://photon.komoot.de/api/?q=berlin

  • Search, with priority to a geo position

    http://photon.komoot.de/api/?q=berlin&lat=52.3879&lng=13.0582

  • Limit number of results

    http://photon.komoot.de/api/?q=berlin&limit=1

  • Force preferred language

    http://photon.komoot.de/api/?q=berlin&lang=de
